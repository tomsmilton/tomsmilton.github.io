Hollybush Crack Bot

Tweets every time there is a new log on Hollybush Crack.

This bot is powered by a Github action that runs four times a day. This action runs a Python script that checks Hollybush Crack on UKC (scraping with BeautifulSoup) to see if there is a new log (data stored in a JSON file in the repo). If there is a new ascent then it tweets about it (using the X API). See the Github repo here.

Job Selector Tool

An app that helps medics rank their job preferences starting with a CSV with all the jobs in.

Every couple of years UK trainee Doctors have to rank their job preferences. They sit there with a list of 100s of jobs in their region and have to decide if they'd like Paediatrics in Doncaster or General Surgery in Barnsley. They're not given any helpful guidance on good systems for doing this and often spend hours agonising over a spreadsheet. This tool takes a CSV of jobs and runs through pairwise ranking to give Doctors a ranking system that is actually helpful. The current version needs updating to work a bit better - a project for an evening soon. I originally wrote this to help my girlfriend rank her jobs - I couldn't deal with watching more spreadshseet shenanigans.

Deanery Ranker

Rank the UK medical deaneries to work out your application preferences.

At the end of medical school UK doctors have to rank the regions of the UK. They take a list of deanearies and stack them up in order of preference. Again they are given no helpful guidance on how to do this and often spend hours agonising over a spreadsheet. This tool has all the deaneries pre-loaded and allows you to quickly go through a pairwise comparison.

Image measurement tool

Upload an image and measure distances, draw shapes, and calculate areas with a reference scale.

When moving house I wanted a quick way to draw to-scale furniture onto the floorplan of my new rooms. I could open up Photoshop and count pixels - or use this simple vibe coded tool to do it.

Flora's bins

What is the next bin?

Sheffield City Council have a helpful website for residents to check which bin will be collected next. Whilst it works well it is far too many clicks. I just want to open a page and see the colour of the next bin. This tool uses a Github action once a day to scrape the council's website and then presents a simple interface with the next bin.
